# CV Builder AI — Xây Dựng CV Thông Minh Cùng AI

**CV Builder AI** là ứng dụng giúp bạn tạo ra một bản CV chuyên nghiệp, ấn tượng chỉ trong vài phút — có trợ lý AI đồng hành từng bước, hiểu tiếng Việt lẫn tiếng Anh.

> Không cần biết thiết kế. Không cần biết viết CV. Bạn chỉ cần trò chuyện với AI, mọi thứ sẽ được làm cho bạn.

---

## 4 Tính Năng Chính

### 1. Chọn Mẫu CV & Tự Chỉnh Sửa

Chọn từ bộ sưu tập mẫu CV đẹp, hiện đại. Sau đó thoải mái chỉnh sửa trực tiếp:

- Thay đổi màu sắc, font chữ, cỡ chữ
- Thêm/xóa/sắp xếp các mục (kinh nghiệm, học vấn, kỹ năng...)
- Xem trước CV ngay lập tức khi chỉnh sửa — giống như đang dùng Word
- Xuất ra file PDF chỉ với 1 nút bấm

### 2. Upload CV Cũ — AI Tự Động "Số Hóa"

Bạn đã có CV cũ dạng PDF hoặc Word? Đừng gõ lại từ đầu:

- Kéo thả file CV cũ vào ứng dụng
- AI sẽ tự động đọc, hiểu và điền mọi thông tin vào mẫu CV mới
- Chỉ mất vài giây — bạn có thể chỉnh sửa lại sau

### 3. Trò Chuyện Với AI Để Sửa CV

Đây là tính năng "đặc biệt" nhất của ứng dụng:

- Mở khung chat, nói chuyện với AI **bằng tiếng Việt**
- Ví dụ: *"Đổi font chữ thành Roboto", "Thêm mục kỹ năng", "Làm phần kinh nghiệm ngắn gọn hơn", "Đổi màu chủ đạo thành xanh dương đậm"*
- CV thay đổi **ngay lập tức** trước mắt bạn — không cần reload trang
- Bạn có thể undo nếu không thích thay đổi

### 4. Tối Ưu CV Theo Mô Tả Công Việc (JD Tailor)

Có một tin tuyển dụng bạn muốn ứng tuyển?

- Dán toàn bộ mô tả công việc (JD) vào ứng dụng
- AI sẽ phân tích yêu cầu của nhà tuyển dụng
- Tự động điều chỉnh CV của bạn để **phù hợp nhất** với vị trí đó
- Hiển thị điểm số match — bạn sẽ biết CV của mình "khớp" bao nhiêu % với JD
- Gợi ý cụ thể những gì cần cải thiện

| Việc bạn làm thủ công | CV Builder AI làm cho bạn |
|---|---|
| Ngồi căn lề, chọn font, phối màu | AI tự làm theo yêu cầu bạn nói |
| Gõ lại từng dòng từ CV cũ | Upload file → AI tự điền |
| Đọc JD dài 2 trang, đoán ý nhà tuyển dụng | AI phân tích JD, so sánh với CV của bạn |

| Tạo nhiều phiên bản cho nhiều công ty | Mỗi JD tạo ra 1 CV tối ưu riêng |

## Bạn Có Thể Dùng AI Nào?

Ứng dụng hỗ trợ **6 nhà cung cấp AI** khác nhau. Bạn có thể chọn bất kỳ AI nào bạn có tài khoản:

| AI | Ghi chú |
|---|---|
| **Claude** (Anthropic) | Khuyên dùng — hiểu tiếng Việt tốt nhất |
| **GPT-4o** (OpenAI) | Cần API key từ OpenAI |
| **Gemini** (Google) | Cần API key từ Google |
| **Grok** (xAI) | Cần API key từ xAI |
| **DeepSeek** | Cần API key từ DeepSeek |
| **Ollama** | Miễn phí 100% — chạy AI trên chính máy bạn, không cần Internet |

> Nếu bạn không muốn tốn tiền API, chọn **Ollama** — AI chạy ngay trên máy tính của bạn, hoàn toàn miễn phí.

## Giao Diện — Nhìn Là Hiểu

Ứng dụng được thiết kế đơn giản, chia làm 3 vùng chính:

```
┌─────────────┬──────────────────────┬──────────────┐
│             │                      │              │
│  Bảng điều  │   Xem Trước CV       │  Chat AI /   │
│  chỉnh sửa  │   (giống A4 thật)    │  JD Tailor   │
│  (bên trái) │                      │  (bên phải)  │
│             │                      │              │
└─────────────┴──────────────────────┴──────────────┘
```

- **Bên trái**: Bảng điều khiển — đổi màu, font, thêm xóa mục
- **Ở giữa**: CV hiển thị y hệt như khi in ra giấy A4
- **Bên phải**: Khung chat với AI (mở khi cần)

## Một Vài Ví Dụ Khi Chat Với AI

Bạn có thể nói chuyện với AI bằng tiếng Việt như nhắn tin với bạn bè:

```
Bạn: "Đổi tất cả font thành Roboto đi"
AI:  "Đã chuyển toàn bộ CV sang font Roboto. Trông hiện đại hơn rồi đấy!"

Bạn: "Thêm mục Chứng chỉ, trong đó có IELTS 7.5 và TOEIC 900"
AI:  "Đã thêm mục Certifications với 2 chứng chỉ bạn yêu cầu."

Bạn: "Phần kinh nghiệm dài quá, rút gọn mỗi cái còn 3 gạch đầu dòng thôi"
AI:  "Đã rút gọn, mỗi kinh nghiệm còn 3 ý chính."

Bạn: "Đổi màu tiêu đề thành xanh navy #1e3a5f"
AI:  "Đã cập nhật màu tiêu đề sang xanh navy."
```
